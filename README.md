## Overview

This portfolio website was inspired by my internship experience. I used it as an opportunity to recreate and practice modern web design while presenting my projects.

## Features

* Modern and clean UI design
* Hero section as a visual entry point
* Parallax effects for a dynamic feel
* Project showcase
* Contact information
* Responsive layout for different devices

## Technologies Used

* HTML5 for structure
* SCSS for styling (compiled to CSS)
* JavaScript for interactivity

## Purpose

The goal of this project was to experiment with modern design concepts and effects, such as hero sections and parallax scrolling.
